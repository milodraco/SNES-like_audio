□ 概要
========================================

  "SNES SPC700 Player" をお選びいただきまして、ありがとうございます。

  "SNES SPC700 Player" は、スーパーファミコンの音楽ファイル (SPC) を再生するための、
  とても軽量でシンプルな設計の SPC 専用プレイヤーです。


  本ソフトウェアの特徴は以下のとおりです。

    ・ 再現性・音質に優れた Alpha-II Productions 製の SNESAPU ライブラリ v2.0 をベースに
       独自に改良を加え、高機能・軽量・安定化した再生エンジンを使用しています。

    ・ 保存しただけでは演奏できない特殊な SPC を演奏させる拡張規格 Script700 (SE) の
       互換機能を搭載しています (SNESAPU.DLL 単体では TimerTrick 互換機能もあります)。

    ・ SPC に特化した情報表示機能 (ID666、レジスタの簡易モニタ、グラフィック インジケータ)
       などを搭載しています。 ID666 はテキスト形式、バイナリ形式の両方に対応しています。

    ・ ID666 による演奏時間、フェードアウトの設定が利用できます。 設定されている演奏時間を
       無視したり、エンドレス再生も可能です。 シーク (早送りや巻戻し) にも対応しています。

    ・ 無音検出に対応。 効果音のように数秒で止まってしまっても一定時間経過後に自動で
       次の曲に移るので、演奏時間の設定がない SPC でも違和感の少ない連続再生ができます。

    ・ SPC 再生の仕組み上、これまで逆シーク (巻戻し) は非常に遅い処理のひとつでしたが、
       キャッシュ機能を搭載し、ほぼ待ち時間なしの逆シークを実現しています。

    ・ お使いのコンピュータ (サウンド機能) とインストールされているシステムが対応していれば
       最大 32bit-float (IEEE754) 出力による高精度な演奏・出力ができます。

    ・ レジストリの書き込みを行わないため、アンインストールはファイルを削除するだけです。
       とりあえずお試しで使いたい方でも、気軽にお使いいただけます。


  本ソフトウェアは以下のオペレーティング システム上で動作するよう設計されています。

    ・ 32bit 版 Windows 2000, XP, Vista, 7, 8, 8.1, 10
    ・ 64bit 版 Windows Vista, 7, 8, 8.1, 10 (※)

    ※ 64bit 環境では WOW64 機能により 32bit 版として動作します (Windows Vista, 7 では
       タスクマネージャで "*32" が、WIndows 10 では "32ビット" が表示されます)。


  Linux 環境では Wine (http://www.winehq.org/) をインストールすることで Windows 専用の
  ソフトウェアを動作させることができるようになります。 "SNES SPC700 Player" の場合は
  多少の不具合が確認されていますが、通常の演奏は行えるようです。

  ただし、以下の重大な不具合が確認されていますので、ご使用の際は十分ご注意ください。

    ・ Wine で動作させている環境で [ビット] メニューから [24 ビット] [32 ビット] を
       選択するときは音量にご注意ください。 Wine のサウンド ドライバが 24 ビット以上の
       出力に対応していない場合、大音量のノイズが出力されることがあります。


□ インストール方法
========================================

  SPCPLAY.EXE, SNESAPU.DLL を同じ場所に置いてください。
  あとは SPCPLAY.EXE をダブルクリックするだけで起動できます。

  新しいバージョンに更新する場合は、古いファイルを新しいファイルで上書きしてください。
  SPCCMD.EXE は SPC のデバッグに使用します。 SPC ファイルを演奏するだけであれば不要です。
  削除してしまってもかまいません。

  ファイルの説明
  ~~~~~~~~~~~~~~
  ・ README.TXT  … このファイル
  ・ SPCPLAY.EXE … SNES SPC700 Player 本体
  ・ SNESAPU.DLL … SNES SPC700・DSP のエミュレータ
  ・ SPCCMD.EXE  … SNES SPC700 Player デバッグツール (※)

  ※ SPCCMD.EXE はコンソール用です。 コマンドプロンプトから実行してください。


□ アンインストール方法
========================================

  SPCPLAY.EXE, SNESAPU.DLL, SPCCMD.EXE, README.TXT, SPCPLAY.INI (※), SPCPLAY.STK (※)、
  作成した不要なプレイリスト ファイル (*.LST) を削除します。

  ※ SPCPLAY.INI と SPCPLAY.STK は SPCPLAY.EXE と同じ場所に自動作成されます。

  レジストリの書き込みやシステム ファイルの変更などは行っていませんので、ファイルの削除
  のみでアンインストール完了です。
  お使いいただき、ありがとうございました。


□ 基本的な使い方
========================================

  演奏を開始する
  ~~~~~~~~~~~~~~
    SPCPLAY.EXE を起動したら、 [OPEN] ボタンをクリックして SPC ファイルを開いてください。
    ファイルが開かれると自動的に演奏を開始します。 ファイルをウィンドウ上にドロップして
    演奏を開始することもできます。
    圧縮ファイルの場合はいったん解凍して SPC ファイルを取り出した後に開いてください。

  演奏を停止する
  ~~~~~~~~~~~~~~
    [PAUSE] ボタンをクリックすると一時停止、 [STOP] ボタンをクリックすると完全に停止
    できます。 再開する場合は [PLAY] ボタンをクリックします。

  プレイリストにファイルを登録する
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    [APPEND] ボタンをクリックすると、現在開かれているファイルがプレイリストに追加され、
    ファイルのタイトルが表示されます。
    ファイルをプレイリスト上にドロップして追加することもできます。

  プレイリストのタイトルを削除する
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    [REMOVE] ボタンをクリックすると、選択しているプレイリスト中のタイトルを削除できます。
    [CLEAR] ボタンをクリックすると、すべてのタイトルを削除できます。
    どちらもプレイリストから登録を消すだけで、 SPC ファイル自体は削除されません。

  プレイリストに登録したファイルを演奏する
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    プレイリストから聴きたいタイトルをダブルクリックするか、選択後 Enter キーを押します。

  プレイリストを保存する
  ~~~~~~~~~~~~~~~~~~~~~~
    [SAVE] ボタンをクリックしてください。 自由にファイル名を付けることができます。

  プレイリストを読み込む
  ~~~~~~~~~~~~~~~~~~~~~~
    [OPEN] ボタンをクリックしてプレイリスト ファイルを開きます。 ファイルをウィンドウ上に
    ドロップして開くこともできます。

  エンドレス再生を行う
  ~~~~~~~~~~~~~~~~~~~~
    [設定] - [演奏時間] の順にメニューを開き、 [有効] にチェックがある場合はクリックして
    チェックを外してください。 タイムゲージが消え、エンドレス再生モードになります。

  キーボード ショートカットを使う
  ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    SNES SPC700 Player では、以下のキーボード ショートカットが利用できます。
    ある程度の操作はキーボードだけでできるようになっています。

    ・ ←                 : インジケータ表示切り替え (前へ)
    ・ →                 : インジケータ表示切り替え (次へ)
    ・ ↑                 : プレイリストのタイトル選択 (上へ)
    ・ ↓                 : プレイリストのタイトル選択 (下へ)
    ・ Shift              : 別機能の呼び出し
    ・ Enter              : プレイリスト選択中のタイトルを演奏
    ・ Ctrl + Enter       : プレイリストの中からランダムで演奏
    ・ Ctrl + O           : ファイルを開く
    ・ Ctrl + S           : プレイリストを保存
    ・ Ctrl + P           : 演奏開始／一時停止
    ・ Ctrl + R           : 最初から演奏
    ・ Ctrl + T           : 演奏停止
    ・ Ctrl + Z           : 前の曲へ
    ・ Ctrl + X           : 演奏開始
    ・ Ctrl + C           : 一時停止
    ・ Ctrl + V           : 演奏停止
    ・ Ctrl + B           : 次の曲へ
    ・ Ctrl + ↑          : プレイリストの項目を移動 (上へ)
    ・ Ctrl + ↓          : プレイリストの項目を移動 (下へ)
    ・ Ctrl + ←          : 演奏速度 (遅く)
    ・ Ctrl + →          : 演奏速度 (速く)
    ・ Ctrl + , (<)       : 開始位置を設定 (※)
    ・ Ctrl + . (>)       : 終了位置を設定 (※)
    ・ Ctrl + / (?)       : 位置をリセット (※)
    ・ Shift + Enter      : プレイリストの中からランダムで演奏
    ・ Shift + ↑         : 前の曲へ
    ・ Shift + ↓         : 次の曲へ
    ・ Shift + ←         : シーク (前へ)
    ・ Shift + →         : シーク (後へ)
    ・ Alt + ↑           : プレイリストの項目を移動 (上へ)
    ・ Alt + ↓           : プレイリストの項目を移動 (下へ)
    ・ Alt + ←           : 音量 (小さく)
    ・ Alt + →           : 音量 (大きく)
    ・ Insert             : プレイリストに追加
    ・ Shift + Insert     : プレイリストに挿入
    ・ Delete             : プレイリストから削除
    ・ Ctrl + Delete      : プレイリストをクリア
    ・ Pause/Break        : 音色番号の 10 進数・ 16 進数表示切り替え
    ・ F1 〜 F8           : チャンネル マスク切り替え (チャンネル別)
    ・ Ctrl + F1 〜 F8    : チャンネル マスク切り替え (ソロ)
    ・ F9                 : チャンネル マスク切り替え (全反転)
    ・ Ctrl + F9          : チャンネル マスク切り替え (全有効)
    ・ Esc                : 最小化
    ・ テンキー 0         : 演奏停止
    ・ テンキー 1         : 演奏速度 (遅く)
    ・ テンキー 2         : 次の曲へ
    ・ テンキー 3         : 演奏速度 (速く)
    ・ テンキー 4         : シーク (前へ)
    ・ テンキー 5         : 演奏開始／一時停止
    ・ テンキー 6         : シーク (後へ)
    ・ テンキー 7         : 左右反転度 (混合側へ)
    ・ テンキー 8         : 前の曲へ
    ・ テンキー 9         : 左右反転度 (反転側へ)
    ・ テンキー /         : フィードバック反転度 (標準側へ)
    ・ テンキー *         : フィードバック反転度 (反転側へ)
    ・ テンキー -         : 音量 (小さく)
    ・ テンキー +         : 音量 (大きく)
    ・ 各文字キー         : プレイリストの先頭文字を検索 (全角文字は不可)

    ※ [設定] - [演奏順序] で [演奏停止] [リピート] が選択されている場合のみ機能します。


□ オススメの設定
========================================

  基本的には、デフォルトの設定がオススメの設定になっています。
  お好みに合わせて以下の設定を変更することにより、より聴きやすい音質になると思います。

  設定例：
  ・ [ビット] を [32 ビット (float)] に設定する。
  ・ [補間処理] を [シンク関数補間] に設定する。
  ・ [拡張設定] の [実機ローパス フィルタ] をオンにする。
  ・ [拡張設定] の [サラウンド無効] をオンにする。

  ※ [サンプリング レート] はできるだけ [32,000Hz] のまま変更しないでください。
     実機の出力仕様が 32,000Hz のため、それ以外のサンプリング レートを使用すると、
     SNESAPU 内部で周波数コンバータが働き、実機と異なる音になることがあります。
